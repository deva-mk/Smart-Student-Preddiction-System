# 🎓 Smart Student Performance Predictor

An AI-powered machine learning application that predicts student academic performance based on various factors such as study habits, attendance, parental involvement, and previous academic records. The project helps educators identify students who may need additional support and enables data-driven educational decisions.

---

## 📌 Project Overview

The Smart Student Performance Predictor uses Machine Learning algorithms to analyze student data and predict future academic outcomes. The system provides valuable insights into the factors influencing student success and helps institutions improve student performance through early intervention.

---

## 🚀 Features

* 📊 Predicts student academic performance using Machine Learning.
* 🧹 Performs data cleaning and preprocessing automatically.
* 📈 Provides data visualization and performance insights.
* 🎯 Identifies students at risk of poor academic performance.
* 🤖 Uses predictive analytics for better educational planning.
* 📉 Evaluates model performance using standard metrics.
* 💻 Interactive and user-friendly implementation using Python.

## 📂 Dataset

The dataset contains approximately **40,000 student records** with multiple features affecting academic performance.

### Example Features:

* Gender
* Age
* Study Hours
* Attendance Percentage
* Previous Grades
* Parental Education Level
* Internet Access
* Extracurricular Activities
* Family Support
* School Type

### Target Variable:

* Final Grade / Student Performance Score
* Pass or Fail Classification

---

## 📊 Model Performance

### Regression Task

* Model: Linear Regression
* R² Score: **0.72**
* Objective: Predict final student grades.

### Classification Task

* Model: Logistic Regression
* Accuracy: **52%**
* Objective: Predict whether a student will pass or fail.
